To create our own API's. We need to install:
1. NodeJS
	npm (node package manager) - come auto with NodeJS
2. json-server



How to create your own API's?
1. To run API. First we need to have a json file, which contain some data.
Because any API works basically on data [if you have data API can get/post/delete data].
the file can be any name but (.json) extension.

-- [a .json file is created on Notepad] 

2. Now, to create your own API. Go to the location where you put the .json file

3. in the path write (cmd) & press Enter.
Currently your path is pointing to your location whereever th json file available

4. write (json-server + json name) -> ex. `json-server students.json` in the command, Enter.
	- It will start create the student API.
	- take the Endpoint → http://localhost:3000/students
	- now you can open the link in the browser & in the postman and do some vaildation (TC are wrote in Excel file)

Note: whenever you close the window, the API will stop running. [just minimize the window].

-> Get http://localhost:3000/students/3
return student id = 3

-> Get http://localhost:3000/students/
return all students

-> Post http://localhost:3000/students/
create a new student
note: you don't have to spcify the id in the link
note: put new student data in Body - raw - json 

-> Put http://localhost:3000/students/5
update student info
note: put new student data in Body - raw - json 

-> Delete http://localhost:3000/students/5
delete student

We send & recieve data in json format

what is JSON? When to use?

- JSON (Java Script Object Notation)
  used whenever you’re communicating between the client and server, 
 data transform through the network in JSON or XML format, because:
    - low weighted
    - able to encrypted/decrypted.

- JSON is preferable. 
- JSON internet Media type is application/json.

- what type of data you are gonna get response? 
based on Media type you know if it is JSON or XML.

- JSON data types is represented as (key: value)
- JSON data types.
    1. String
        - string is must written in double quotation
        - ex. {"name": "John"}
    2. Number
        - must be on integer or a floating
        - no need to " "
        - ex. {"age": 30}
    3. Boolean
        - ex. {"Status":true}
    4. Object
        - ex. "employee":{"name":"John", "age":30}
    5. Array
        - ex. "employee":{"name":"John", "age":30, "Phone": [12344,556789]}
    6. Null
        - ex. {"Secondname":null}`
- Notes:
    - separate every value with a comma.
    - curly brackets means objects
    - square brackets means arrays
