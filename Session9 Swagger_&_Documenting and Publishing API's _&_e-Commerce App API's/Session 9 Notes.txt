# Swagger 

* swagger example [JSON]

Go to → https://petstore.swagger.io/		- 	TO user model

Import curl of
 	POST 	https://petstore.swagger.io/v2/user

In Body/Json, we want to pass the data dynamically

	{
	  "id": "{{id}}",
 	 "username": "{{user_name}}",
  	"firstName": "{{firstName}}",
  	"lastName": "{{lastName}}",
  	"email": "{{Gmail}}",
  	"password": "{{password}}",
  	"phone": "{{phone}}",
  	"userStatus": "{{userStatus}}"
	}

In pre-script we create random string and int variables 


	var randomString = Math.random().toString(36).substring(2, 8);
	var randomInt = Math.floor(Math.random() * 10000);

	pm.collectionVariables.set("id",randomInt);
	pm.collectionVariables.set("user_name",randomString);
	pm.collectionVariables.set("firstName",randomString);
	pm.collectionVariables.set("lastName",randomString);
	pm.collectionVariables.set("Gmail",(randomString+"@gmail.com"));
	pm.collectionVariables.set("password",randomString);
	pm.collectionVariables.set("phone",randomString);
	pm.collectionVariables.set("userStatus",randomInt);


Now chaining & import the rest of req-> Get User   ,  Update User   ,  delete User 


Get	https://petstore.swagger.io/v2/user/{{user_name}}

Put	https://petstore.swagger.io/v2/user/{{user_name}} 	- Note: update dynamically

Del	https://petstore.swagger.io/v2/user/{{user_name}}

then to → POST /pet

-

convert : `Parameter content type` & `Response content type`
	to `application/xml`
-

import its curl

-

Body is [i change <Category>/name & <name>]

<?xml version="1.0" encoding="UTF-8"?>
<Pet>
	<id>0</id>
	<Category>
		<id>0</id>
		<name>Cat</name>
	</Category>
	<name>Jasper</name>
	<photoUrls>
		<photoUrl>string</photoUrl>
	</photoUrls>
	<tags>
		<Tag>
			<id>0</id>
			<name>string</name>
		</Tag>
	</tags>
	<status>available</status>
</Pet>
